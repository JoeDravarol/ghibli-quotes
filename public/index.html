Ghibli Quotes

A free restful API serving quality quotes from Studio Ghibli's film

Try it

Run this code to get a random quote

      
        fetch('https://ghibli-quotes.herokuapp.com/api/random')
          .then(response => response.json())
          .then(quote => console.log(quote))
      
      
      
        {}

Get all quotes

      
        fetch('https://ghibli-quotes.herokuapp.com/api/quotes')
          .then(response => response.json())
          .then(quotes => console.log(quotes))
Output 
      
        [{ film: "...", quote: "..."}, ...]
